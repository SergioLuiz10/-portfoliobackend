# Stack Técnica — Sérgio Nunes

## Linguagens de programação

A linguagem principal do Sérgio é o Python, usada na maior
parte dos projetos dele, especialmente nos que envolvem IA
generativa e backend. Sérgio também tem experiência com Java,
que usou junto do framework Spring Boot no desenvolvimento de
bots de WhatsApp. Além disso, trabalha com SQL para consultas
e modelagem de dados.

## IA generativa e LLMs

Esta é a área de maior especialização do Sérgio. Ele tem
experiência prática com:

- Pipelines RAG (Retrieval-Augmented Generation): construção
  completa, do chunking de documentos à geração de respostas.
- LangChain como framework de orquestração de aplicações com
  LLMs.
- Modelos da OpenAI, incluindo modelos de chat (família GPT)
  e modelos de embedding.
- GPT-4o Vision para análise e interpretação de imagens.
- Arquiteturas multi-agente, com orquestração de agentes
  especializados.
- Bancos de dados vetoriais, com destaque para o PGVector.

## Backend e APIs

Sérgio constrói APIs principalmente com FastAPI, o framework
Python que ele mais usa. Tem experiência com desenvolvimento
de endpoints, integração com bancos de dados e estruturação
de serviços. Também trabalhou com Java e Spring Boot em
projetos de backend.

## Bancos de dados

Sérgio trabalha com PostgreSQL como banco relacional, e com a
extensão PGVector para armazenamento e busca de vetores em
aplicações de IA. Usa o Supabase como plataforma de banco de
dados e backend, incluindo o desenvolvimento de Edge
Functions. Tem familiaridade com SQL para consultas e
manipulação de dados.

## Automação de processos

Sérgio tem forte experiência com automação. Trabalha com n8n
para orquestração de fluxos e arquiteturas de agentes, e com
Microsoft Power Automate e Microsoft Forms para automação de
processos corporativos. Já integrou esses fluxos com sistemas
de gestão como o Omie (ERP) e ferramentas como o ClickUp.

## Infraestrutura e DevOps

Sérgio usa Docker para containerização das aplicações e
Docker Compose para orquestrar ambientes locais. Tem
experiência com deploy em produção usando EasyPanel em VPS
(Hostinger). Trabalha com Git para versionamento de código e
GitHub para hospedagem de repositórios.

## Integrações e mensageria

Sérgio já trabalhou com integração de canais de mensageria,
usando a EvolutionAPI para integração com WhatsApp e o
Chatwoot como plataforma de atendimento ao cliente.
